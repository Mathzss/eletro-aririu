## 📌 Sobre o Projeto

Este é o site oficial da **Eletro-Aririu**, desenvolvido para fornecer informações essenciais sobre a oficina. O projeto foi pensado para ser leve, responsivo e funcional, servindo como o cartão de visitas digital da empresa em Aririu, Palhoça/SC.

O grande diferencial é a facilidade de navegação, permitindo que o cliente conheça os serviços e localize a oficina rapidamente através da integração com mapas.

---

## ✨ Funcionalidades

* **Interface Responsiva**: Otimizado para visualização em desktops, tablets e smartphones.
* **Catálogo de Serviços**: Exibição clara das especialidades da oficina.
* **Localização Inteligente**: Integração com a **Google Maps API**, permitindo que o usuário visualize o local exato e trace rotas diretamente pelo navegador.
* **Botões de Ação Dinâmicos**: Atalhos para facilitar a jornada do usuário até a oficina.

## 🗺️ Integração com Google Maps

Para a funcionalidade de geolocalização, o projeto consome a **Google Maps API**. 

> [!IMPORTANT]
> Caso queira rodar este projeto localmente, é necessário configurar uma `API_KEY` válida no arquivo `.env` e habilitar a **Maps JavaScript API** no Google Cloud Console.
